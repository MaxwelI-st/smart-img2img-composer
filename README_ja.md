## 🌟 v1.1 の新機能
v1.1では、UIの整理、専用の在庫管理システム、および大幅に拡充された特殊タグライブラリが導入されました。

### 📦 在庫管理専用タブ
ランダム選出の「偏り」を完全にコントロール。独立したタブにより、すべてのLoRAやワイルドカードアセットの使用回数を監視できます。使用頻度の低い項目を優先することで、数百枚の生成でも多様性を維持します。リアルタイムの在庫確認と詳細なリセットオプションを備えています。

### 🚀 img2img 連携の強化
解析と生成のワークフローがさらにスムーズになりました。「img2imgへ送信」ボタンをクリックするだけで、**生成されたプロンプトとソース画像の両方**を直接 img2img タブへ転送。手動でのファイルアップロードが不要になりました。

### 🔞 拡充された NSFW & フェティッシュタグ
リクエストに基づき、マシン系（搾乳機等）、体液、詳細なフェティッシュ状態など、特殊なタグ定義を大幅に追加しました。**📏 グループ別タグ制限**（基本、人物、NSFW）により、プロンプトが長くなりすぎるのを防ぎ、バランスを保ちます。

### 🧱 インテリジェントなモザイク自動付与
アップロードされた画像内のモザイクや修正を自動的に検出し、重み調整されたモザイク用タグ（薄い/普通/厚い）を注入します。

## 🛠️ 強力なコアシステム

### 1. 📂 高度なプリセット管理
フォルダパス、しきい値、解像度などの複雑な設定を「プリセット」として保存・読み込み。プロジェクトやキャラクターの切り替えが1クリックで完了します。

### 2. 🧬 LoRA一括ウェイト調整
有効なすべてのLoRAのウェイトを、0.05刻みのグローバルスライダーで一括調整。完璧なデザインバランスを維持します。

### 3. 📁 インテリジェントな出力整理
生成された画像をプリセット名、セクション名、または日付に基づいてサブフォルダに自動整理。出力先の混乱を永久に解消します。

## ✨ 最先端の自動プロンプト生成 (WD14 Tagger)

WD14 Tagger との深い統合により、画像から最適なプロンプトを抽出します。

- **カテゴリ別スマート抽出**: 構図、ポーズ、照明、NSFWなど、タグを精密にフィルタリング。
- **一括カテゴリ切り替え**: 1クリックで特定のカテゴリタグを一括選択・解除。
- **カスタム辞書機能**: 抽出された生タグを、独自の記述スタイルや高詳細なフレーズに自動変換。

## 🎲 シームレスな img2img 統合

A1111の img2img タブ下部のアコーディオン内にネイティブに配置。ADetailer や ControlNet との相性も抜群です。

- **ランダム注入スロット**: キャラクター、シチュエーション、および3つのワイルドカードスロットを管理。
- **戦略的プロンプト配置**: ランダムプロンプトを既存プロンプトの「前」か「後」のどちらに挿入するか制御可能。

## 📖 クイックスタートガイド

UI内の「**📖 使い方 (User Manual)**」タブに詳細なマニュアルが組み込まれています。

1.  **設定**: 「⚙️ 設定」タブで画像フォルダとメモファイルのパスを設定。
2.  **確認**: プリセットを保存し、ヘルスチェックが **✅** であることを確認。
3.  **生成**: img2img タブで拡張機能を有効にして「Generate」を押す。
